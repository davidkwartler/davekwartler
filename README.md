# davidkwartler.com

Personal portfolio and resume site.

**Live:** [www.davidkwartler.com](https://www.davidkwartler.com)

## Features

- Animated aurora gradient hero built with Motion
- Career timeline with experience highlights
- Full resume with skills, certifications, and education
- Responsive design with mobile navigation

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Motion (Framer Motion)
- Deployed on Vercel
